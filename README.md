The folder `manga/` contains all the image files. Including the original four-panel comics, as well as comic image data after swapping adjacent panels or randomly shuffling panels. Need to download, 
link: https://drive.google.com/file/d/1umWrd1G_9juL5eXeCXrSPivQ0gavFxvz/view?usp=sharing

The folder `swap/` contains all the text information of the manga strips after the panels are swapped.

The folder `text/` contains the text descriptions and sentiment information of the manga strips.
